## 📐 Tips for Each Generator

### Midjourney
- Add `--ar 16:9` or `--ar 4:3` at the end for aspect ratio
- Add `--v 6` for the latest model
- Add `--style raw` for more photorealistic results
- Example: `[prompt text] --ar 16:9 --v 6 --style raw`

### DALL-E 3 (ChatGPT)
- Specify the aspect ratio in the prompt text (e.g., "16:9 aspect ratio")
- Request "ultra-realistic photography" for best results
- Use the "HD" quality option when available

### Stable Diffusion (e.g., via Automatic1111 or ComfyUI)
- Use a photorealistic model like **Realistic Vision** or **Juggernaut XL**
- Negative prompt: `cartoon, illustration, anime, painting, drawing, low quality, blurry, deformed`
- Recommended settings: CFG 7, Steps 30-40, Sampler DPM++ 2M Karras
